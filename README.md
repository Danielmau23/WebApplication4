# WebApplication4

La aplicación funciona correctamente.

Como ejecutar la aplicación:
 - se tiene que ejecutar el scripBD en sql server para generar la BD.
 - se deben cambiar los credenciales de acceso a la BD desde el codigo de la aplicación "WebApplicatio4", específicamente en el archivo llamado BaseDatos, la variable a cambiar es "credenciales".
 - ahora solo queda ejecutar el programa :)


NOTAS MUY IMPORTANTES
Dado a los permisos que se necesitan para poder usar los correos electrónicos solo investigue como hacerlo con gmail.

Se debe de hacer de esta manera:

- Al registrarse en la página se debe poner el correo gmail y la contraseña con la que se accede a este desde la página de Google, el nombre de usuario puede ser el que usted desee.

- Debe de iniciar sección con esa cuenta en Google, después debe de acceder al siguiente link: https://www.google.com/settings/security/lesssecureapps
y activar el acceso a aplicaciones menos seguras, esto se tiene que hacer porque si esta desactivado no se va a enviar los correos.
Mas info: https://www.youtube.com/watch?v=i5O6rtkeozY

- después de hacer realizado los pasos anteriores ya puede enviar plantillas y mensajes a cualquier correo.

- si desea después de probar la app puede desactivar otra vez los permisos de acceso de aplicaciones menos seguras.
